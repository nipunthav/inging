Inging Rubcase - Simplified Version

How to use:
1. Open index.html in a browser, or upload it to GitHub Pages.
2. Fill in fields on the left.
3. The generated LINE message appears on the right.
4. Click Copy and paste into LINE.

Output format:

รับใหม่ อ.[teacher name]
Ward: [ward name]
ผู้ป่วย: [patient name] [Male/Female] [age]y
HN: [hn]
U/D: [ud]
Problem List:
[problem list]
Plan:
[plan]
